# Проект: Распознавание картинок с использованием TensorFlow

## Описание проекта
Этот проект представляет собой систему распознавания изображений, где пользователи могут регистрироваться, входить в свои аккаунты, загружать изображения для анализа и получать результаты классификации. Проект включает функционал управления учетными записями, удобную домашнюю страницу и дашборд для работы с изображениями.

---

## Функциональность

### **1. Меню дашборта**
Меню дашборта предоставляет пользователю удобный доступ к основным функциям системы. Через меню можно:
- Загрузить изображения для анализа.
- Просмотреть результаты распознавания.

Дашборд доступен только для авторизованных пользователей. Если пользователь не вошёл в систему, он будет перенаправлен на страницу авторизации.

---

### **2. Регистрация**
Функция регистрации позволяет новым пользователям создать свой аккаунт. Для регистрации необходимо:
- Указать адрес электронной почты.
- Придумать логин.
- Придумать и ввести пароль.

После успешной регистрации пользователь автоматически перенаправляется на страницу авторизации.

---

### **3. Авторизация**
Авторизация позволяет зарегистрированным пользователям входить в свой аккаунт. Для входа необходимо ввести:
- Логин.
- Пароль.

После успешной авторизации пользователь получает доступ к дашборду и другим защищённым функциям системы.

---

### **4. Домашняя страница**
Домашняя страница доступна всем пользователям, как авторизованным, так и неавторизованным. Она включает:
- Краткое описание проекта и его возможностей.
- Ссылки на страницы регистрации и авторизации.

---

### **5. Выход из аккаунта**
Функция выхода позволяет пользователю завершить сессию. Для этого:
- В меню навигации есть кнопка "Logout" (Выход), которая видна только авторизованным пользователям.
- После нажатия на кнопку "Logout" пользователь завершает сессию и перенаправляется на домашнюю страницу.

---

## Как запустить
- В **PyCharm**, используя терминал, зайдите в папку проекта.
- Введите в терминале: `python manage.py runserver`.
- Дождитесь запуска проекта и перейдите по ссылке: [http://127.0.0.1:8000/](http://127.0.0.1:8000/), которая отобразится в терминале.
